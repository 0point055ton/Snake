# Snake

Classic game of **Snake** created with Python 3 and the `pygame` module (no pun intended).

---

## Requirements

- Python 3.8 or newer
- `pygame` library
- Installed font from the `resources` folder

---

## How to Play

- Use the **W, A, S, D** keys to move the snake.
- Eat red apples to grow the snake.
- Avoid the boundaries and the snakeâ€™s tail.
- When the game is over, press **SPACE** to retry or **ESC** to quit.
